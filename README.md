Flower Recognition System

项目简介:
基于 TensorFlow 和 MobileNetV2 的智能花卉识别系统。

支持：
1、图片识别
2、摄像头实时识别
3、花卉养护建议推荐

技术栈：
Python、TensorFlow、Keras、Flask、HTML、JavaScript

数据集；
TF-Flowers共5类花卉3278张图片
